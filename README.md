# CONTROL DE EMOCIONES

El Ministerio para el Bienestar Emocional (**MBE**), organismo dependiente de la 'Unión Directiva para el Bien Común' (**UDBC**), nos ha pedido a la Organización de Asuntos Informáticos (**OAI**) que desarrollemos un modelo que, dada una foto en la que se vea la cara de un sujeto, reconozca la expresión en la cara del mismo y, según sea esta y las necesidades del momento, determine qué sustancias administrar para corregir la emoción.

Para ello hemos preparado un dataset, "FER2013", y hemos entrenado un modelo CNN. Posteriormente, transformamos una versión del dataset "Affectnet" en un archivo CSV, utilizando el mismo formato que "FER2013", para reentrenar el modelo.
